# 🌦 SkyPulse Weather
SkyPulse Weather is a modern, responsive weather dashboard that provides real-time weather data, 5-day forecasts, and smart insights like commute timing, rain alerts, and temperature analytics — all in a beautiful animated UI.

It uses the OpenWeatherMap API and Chart.js to deliver interactive weather visualizations with dynamic themes based on weather conditions.

---

## ✨ FEATURES
- 🌍 Search weather by city
- 🌡 Real-time temperature, humidity, wind & feels-like data
- 🌦 Dynamic weather themes (Sunny, Rainy, Cloudy, Thunder, Snow)
- 📅 5-day forecast
- 🚗 Smart commute suggestion (best time to leave based on rain risk)
- ⚡ Rain & storm alerts
- 📊 Temperature analytics chart (Chart.js)
- 🧭 Wind direction detection
- 🗺 Live weather radar (OpenWeatherMap)
- 📱 Fully responsive design
